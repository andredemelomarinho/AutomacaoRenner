História de Usuário 1:
Título: Baixar documentos enviados pelo preposto
Como administrador
Quero poder baixar os documentos enviados pelo preposto
Para revisar e analisar o conteúdo offline

Critérios de Aceite:
O administrador deve visualizar uma lista de documentos enviados pelo preposto.
Cada documento deve ter uma opção de download.
O download deve iniciar ao clicar no botão correspondente.
O nome e o formato do arquivo devem ser mantidos após o download.

História de Usuário 2:
Título: Aprovar documentos enviados pelo preposto
Como administrador
Quero aprovar os documentos enviados pelo preposto
Para que eles sejam validados e sigam para os próximos passos

Critérios de Aceite:
O administrador deve visualizar uma opção de "Aprovar" ao lado de cada documento.
Ao clicar em "Aprovar", o status do documento deve ser atualizado para "Aprovado".
O sistema deve exibir uma mensagem de confirmação após a aprovação.

História de Usuário 3:
Título: Reprovar documentos enviados pelo preposto com justificativa
Como administrador
Quero reprovar documentos enviados pelo preposto e adicionar comentários explicando o motivo
Para que o preposto entenda a razão da reprovação e possa corrigir o documento

Critérios de Aceite:
O administrador deve visualizar uma opção de "Reprovar" ao lado de cada documento.
Ao clicar em "Reprovar", o sistema deve solicitar um comentário obrigatório explicando o motivo da reprovação.
O sistema deve impedir a reprovação sem o preenchimento do comentário.
Após a reprovação, o status do documento deve ser atualizado para "Reprovado".
O preposto deve ser notificado sobre a reprovação junto com o comentário.
