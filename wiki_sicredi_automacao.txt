BDD
Dado que o usuário está logado como Administrador ou Gestor no portal ANS
Quando o usuário acessa a página com a listagem dos fornecedores, o Administrador e gestor e dentro acessa os indicadores daquele fornecedor.
E admin visualizar todos os fornecedores
E gestor visualizar somente o forncedor vinculado ao seu perfil
Então a listagem dos fornecedores redireciona para a lista de indicadores daquele fornecedor
E a visualização é por projeto por fornecedor (número e contrato e status)"prévia","concluído-aprovado","concluído-reprovado", "não enviado"


Cenário 1: Acesso à listagem de fornecedores como Administrador
Dado que o usuário está logado como Administrador
Quando o usuário acessa a página de listagem de fornecedores
Então o sistema deve exibir todos os fornecedores cadastrados
E ao selecionar um fornecedor, o usuário é redirecionado para a página de indicadores desse fornecedor
E a visualização deve apresentar os projetos por fornecedor com as seguintes informações: número, contrato e status ("prévia", "concluído-aprovado", "concluído-reprovado", "não enviado")

Cenário 2: Acesso à listagem de fornecedores como Gestor
Dado que o usuário está logado como Gestor
Quando o usuário acessa a página de listagem de fornecedores
Então o sistema deve exibir apenas o fornecedor vinculado ao perfil do gestor
E ao selecionar o fornecedor, o usuário é redirecionado para a página de indicadores desse fornecedor
E a visualização deve apresentar os projetos por fornecedor com as informações: número, contrato e status ("prévia", "concluído-aprovado", "concluído-reprovado", "não enviado")

Cenário 3: Validação de redirecionamento para indicadores
Dado que o usuário está logado como Administrador ou Gestor
Quando o usuário clica em um fornecedor na lista
Então o sistema deve redirecionar o usuário para a página de indicadores do fornecedor selecionado

Cenário 4: Verificação dos status de projetos nos indicadores
Dado que o usuário acessa a página de indicadores de um fornecedor
Quando a lista de projetos for exibida
Então cada projeto deve mostrar seu respectivo número, contrato e status, que pode ser:

"prévia"
"concluído-aprovado"
"concluído-reprovado"
"não enviado"

Cenário 5: Restrições de visualização para Gestores
Dado que o usuário está logado como Gestor
Quando tenta acessar diretamente a página de um fornecedor que não está vinculado ao seu perfil (via URL)
Então o sistema deve bloquear o acesso
E exibir uma mensagem de erro: "Acesso não autorizado para este fornecedor"

Cenário 6: Verificação de permissões após login
Dado que o usuário está logado no portal ANS
Quando o sistema identifica o perfil do usuário (Administrador ou Gestor)
Então o sistema deve garantir que:
Administradores tenham acesso completo à listagem de todos os fornecedores
Gestores tenham acesso apenas ao fornecedor vinculado ao seu perfil

Cenário 7: Acesso sem permissão adequada
Dado que o usuário está logado com um perfil que não é Administrador ou Gestor
Quando o usuário tenta acessar a listagem de fornecedores
Então o sistema deve negar o acesso
E exibir uma mensagem de erro: "Você não tem permissão para acessar esta página."

Cenário 8: Listagem de fornecedores vazia para Gestor
Dado que o usuário está logado como Gestor
E não há fornecedores vinculados ao seu perfil
Quando o gestor acessa a página de listagem de fornecedores
Então o sistema deve exibir uma mensagem: "Nenhum fornecedor vinculado ao seu perfil."

Cenário 9: Indicadores sem projetos cadastrados
Dado que o usuário acessa a página de indicadores de um fornecedor
E o fornecedor não possui projetos cadastrados
Quando a lista de indicadores for carregada
Então o sistema deve exibir a mensagem: "Nenhum projeto cadastrado para este fornecedor."

Cenário 10: Filtros e ordenações nos indicadores
Dado que o usuário está na página de indicadores de um fornecedor
Quando o usuário aplica filtros por status ou ordena por número/contrato
Então o sistema deve atualizar a lista de indicadores de acordo com os filtros e ordenações selecionados
